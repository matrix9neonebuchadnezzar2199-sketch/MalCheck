MalCheck 機能分析（表層 / 動的 / 静的）

表層・動的・静的の3観点で実装済み機能と今後の推奨機能を一覧化する正本です。フロー図・シーケンスの詳細は Feature Analysis を参照。

凡例: ✅ 実装済み / ⚠️ 部分的（要 Docker・要イメージ・hook のみ等） / ❌ 未実装

横断（3フェーズ共通）

機能状態備考
フェーズオーケストレーション✅run_pipeline / run_pipeline_with_intake
フェーズ失敗の分離✅1フェーズ失敗でもレポート生成継続
Intake（ZIP/7z 解凍）✅mau/intake.py、パスワード試行・ネスト上限
子ファイル fan-out✅展開葉ごとに surface / unpack / static / dynamic

親レポート集約 schema 2.1✅intake.children[]、malicious_findings、verdict 昇格
JSON + HTML レポート✅mau/report_generator.py

実行経路: mau.surface_runner → surface コンテナで scripts/remnux/analyze.py（+ format_scanners.py）

中優先（解析深度）

#機能観点理由
